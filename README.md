Script en bash que automatiza el proceso para flashear la rom de stock en motorola G42 (hawao)

**El script está configurado para descargar la ROM para Argentina, si querés otra tienes que modificar el link**


# Uso
```bash
sudo su
```
```bash

```

```bash
./autoStock.sh
```
